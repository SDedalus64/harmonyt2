Azure-Only Testing Commands

Start fresh build:
npx expo start --clear

Start with full cache clear:
npx expo start --clear --reset-cache

Check bundle size:
du -sh src/

Find large files:
find src/ -type f -size +1M

Check if local data removed:
ls -la src/
